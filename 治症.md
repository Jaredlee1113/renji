# 耳

任何耳朵的问题，要想到肾脏。轮刺耳朵前三个穴道：耳门、听宫、听会。久鸣为虚，以补肾为主。
耳鸣：久鸣：肾俞、志室、章门、复溜（要下的好，补针）。实证：然谷、外关、支沟（又称飞虎）。流脓：麝香（无恐不入）矾石（干燥、防腐）散，比例 2:45。

# 面部中风

中风原因：大汗后，吹风。例如，大汗后，对着脸吹冷气。又如，大汗后，开车开窗吹风。
先健侧，后患侧。对侧合谷，地仓透夹车、太阳。嘴巴流涎封口：地仓透到嘴巴里，来回拉扯。脸正后，针人中、承浆，固定住。

# 肺脏

肺主皮毛，主鼻子。肺主呼（吐气）。肺，水之上源。

**虚**：气虚，无力，面色仓，呼吸短促、不渴。下针：太渊、肺俞、中府。强土补气：足三里、中脘、tan 中、脾俞、章门
**实**：热咳、浓痰、面赤、皮干、渴。尺泽、肺俞、中府，针 tan 中、肾俞、金门、然谷。
**胸腔内伤**：肩井、足三里（活血化瘀）。淤血往外导，大便会是血便，黑色的。脏气乱：天柱、大抒。平五脏气乱。
**发癫痫**：发病时：鸠尾（痰的源头）、人中、涌泉。丰隆导痰。没有发病：昼发：灸申脉；夜发：灸照海。不定时发：申脉、照海都灸。除了前面三个：百会、中府、魄户、神庭、劳宫、神门。
**肺癌**：早期：咳清白泡沫痰。下针：肺俞、中府透云门、脾俞、胃俞、中脘、足三里（补胃气）、孔最（㕁穴止痛）。女人肺癌，奶水没有下去。调月经：地机、三阴交。咳嗽：公孙、内关。久咳：隔蒜灸舌头。
**肺痨**：血脉大、血管也变大。灸痨法：四花灸：肺俞、膏肓。灸到肚子里面有咕噜咕噜的水声（尤其是灸膏肓），就代表艾灸的力量到了。

# 心脏

心肺关系：心脏像汽车引擎，肺就像水箱。心脏的问题，要固肺。肺脏的问题，要固心脏。这两个脏气组成**上焦**。
**肺积水**：但坐不能卧。下针：隔俞、tan 中、鸠尾、乳根。导水：水沟、阳谷、沟谷、后溪、照海。外用：黄连一钱、巴豆一颗。巴豆仁去壳，和黄连粉混合，把这个粉饼放到肚脐上，灸七壮，烧到病人肠鸣，病人就会去上厕所排掉。
**胸肋积水**：几乎都在左肋，左肋痛。可以躺下去睡，不能躺左边。下针：外关、临泣、水分、水道、乳根、大包。
**心脏肥大**：心脏会肥大，就是水多了。症状：胸痛，分为刺痛和闷痛。刺痛有淤血，闷痛为气阻。刺痛下针：巨阙、心俞、少府、神门、劳宫、关元。闷痛：公孙、内关、天突、巨阙、关元、第十椎下。淤血气阻分不清的，先下针：公孙、内关、巨阙、关元，下完后基本就去掉了。
**心率不整**：正常人：手脉跳 54 下停一下。少于 54 都不正常。剧痛在正中间，公孙、内关。痛在肾经，对侧然谷，或者用井穴。井主心下满。如果是阴跷脉，照海。如果是胃经，历兑。
**急救心脏病**，胸口还有余温，下少府、关元。少府下了没有跳起来，再下关元。
**乳癌**：成因：奶水没有完全下去。初期在天应灸，灸一下（5、7、9 壮）就好了。如果变大的很大，红肿了，用导引法。下针：肩井、足三里。下针当场硬块就软掉了。这是统治乳房的。之后循经取穴。扎针后，可以在天应上灸。尤其是凹下去的。甚至化脓的都可以灸。近取穴：膺窗、乳根。
**全身脓水**两种：无味和腥臭。无味隔附（生附子）灸，腥臭：面粉、硫磺（石硫黄）、大蒜等量做成面团。灸一次换一个饼，灸 21 壮，把腐烂的肌肉排掉。

# 胆

查证胆结石：阳陵泉下一寸，压痛。第十一椎下，压痛。晚上十一点到一点最难过。
肝和脾发黄，肝为青黄，脾为土黄。分辨不出来，胃口分辨，都没有胃口，肝的话多了一个恶心呕吐。

巨阙、关元、心俞、胆俞。胆堵了，全身发黄，第七椎下至阳穴。最有名的治黄的穴道就是至阳穴。

针灸治病的疗程：隔天。
针灸后的禁忌：

1. 喝酒
2. 辣椒
3. 麻油

# 脾

治疗脾脏，先去固肾，病就不会再进了。

脾脏肿大：脸色发黄、没有胃口、肚子大、身重。

三皇：阴陵泉、地机、三阴交；章门、脾俞。强水：京门、复溜。

# 胃

眉毛有悬针纹，就说明有胃病。

胃痛：

-   寒：舌苔白厚。寒症用灸。
    灸：中脘、公孙、内关、足三里。梅花灸。
-   热：舌苔黄湿。热症下针。
    针：中脘、胃俞、足三里。梅花针。

胃出血：大便黑的，远血。

血症：

-   内出血：不针本穴。
-   外出血：然谷放血。不够的话，三毛、冲阳放血。醒过来如果有痛，针 肩井、足三里。止痛按照经络取血。

朝食暮吐：中脘下针。

腹膜炎：灸肘尖。灸到肠鸣。

# 消渴（糖尿病）

脾脏管胃口，食物吃到嘴巴里面味觉如何是胰脏。
嘴巴里面的津液来源于大肠，大肠的水靠小肠的火蒸发上于肺。
燥渴：天突、关元、天枢、中府、云门、肺俞、第八椎外开一寸半（糖尿病特效穴）、阳池（收针）。

# 痞块（癌）

正面可以摸到不规则、凹凸不平的硬块，为痞块。光滑的是良性的。
阴积造成痞块。所以行阳的速度加快就不会累积了。
痞根+消积。例肝有痞块，灸痞块，下针：肝关、肝一肝二肝三、太冲、曲泉（虚症）、行间（痛的厉害）、肝俞、期门（俞募治疗）。
痞块出现原则：固胃，让胃口很好，同时行阳气，让气机流畅。

# 肝

中医不赞成把胆拿掉，如果胆没了，（大概十五年）胆汁会聚集在总胆管，最终会造成肝结石。
任何的肋间痛，都可以在对面的肝关下针止痛。天枢、期门、日月、肝俞、胆俞。痛的很厉害，直接在上面灸。

# 腹痛

淤血造成的腹痛：月经、被人踢到、腹部出血等。怎么判断：左腹关元外开三寸的地方有压痛点。人身左边为血，右边为气。

-   便秘腹痛
    -   照海、支沟（又名：飞虎），专治便秘，不分大小肠，不分寒热。
    -   寒：不上厕所不难过。灸：天枢、关元、中脘 =》四花灸
    -   热：难过。
    -   小肠堵到，会一直放屁。强心让热进入小肠，强脾让湿进入小肠，让津液进入小肠。关元、巨阙、章门、照海、支沟、太白。
    -   堵在大肠头，少腹隐隐做痛，足三里下一寸阑尾穴下针，关元、天枢、中脘、支沟、照海。
    -   盲肠炎。盲肠炎的起因，一次吃大量东西，吃完又跑来跑去运动。足三里下一寸（阑尾），会有压痛点，会有一个像绿豆一粒，下针的同时病人的痛就消失了。
    -   堵到肚脐旁边：天枢有压痛点。下针：支沟、照海、中极、巨阙。
    -   堵到大肠拐弯处，收到惊吓后，造成燥矢堵到大肠的那根血管，造成奔豚。遇到奔豚的时候，要强心脏，让心脏加强力量，把燥矢浊气逼回大肠。巨阙、心俞、少府、神门、内关、劳宫=》强心；再加治疗便秘的穴道：天枢、中脘、关元、支沟、照海。

# 奔豚

堵到大肠拐弯处，收到惊吓后，造成燥矢堵到大肠的那根血管，造成奔豚。遇到奔豚的时候，要强心脏，让心脏加强力量，把燥矢浊气逼回大肠。巨阙、心俞、少府、神门、内关、劳宫=》强心；再加治疗便秘的穴道：天枢、中脘、关元、支沟、照海。

# 痔疮

承山、孔最、长强下针。出血很多后，见下：
肠风下血：从尾椎骨到第十四椎，一路往上按，有痛点，差不多每隔一个关节会有一个痛点，最常见的三个椎，在痛点隔姜灸。专门治肠漏、大便下血用，可能是内痔、大肠炎，大便带鲜血。
